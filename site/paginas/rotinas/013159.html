Importação de Coordenadas para Geoposicionamento

Rotina 013159

Resumo

    Essa rotina serve para fazer a associação da localização com o PDV. O objetivo dela é fazer a atualização, pois as vezes o sistema do RN não funciona para fazer a validação, ou a solicitação so esta sendo feita para atualização da localização do PDV.

    Para atualizar deve ser seguido os seguintes passo:

  1. Primerio coletar as informações nos grupos, onde os RN solicitam a atualização do GPS.

    2. As informações consistem em: a) O codigo do PDV; b) A coordenada dalela localização
    A coordenada tem o padrão : -xx.xxxxxx, -xx.xxxxxx, onde elas podem ter mais casas decimais, aumentando a precisão, porem para a atualização é necessario somente as 6 primerias casas decimais.

  2. Apos coletar as informações, deve ser carregada no arquivo Ajuste Coordenada - Semana X.xlsx.

    3. Esse arquivo vai ter uma função que irar retornar os valores formatados do PDV junto da coordenada.

  3. Colocar o valor no arquivo GPS.CSV, e carregar esse arquivo na dvs.

  4. Por fim, deve ir na rotina 01.31.59, e na caixa de dialogo escrever o nome do arquivo(GPS.CSV), e confirmar.

Detalhado

  1. Primerio coletar as informações nos grupos, onde os RN solicitam a atualização do GPS.

    2. As informações consistem em:

    1. O codigo do PDV;
    2. A coordenada da localização.

    A coordenada tem o padrão : -xx.xxxxxx, -xx.xxxxxx, onde elas podem ter mais casas decimais, aumentando a precisão, porem para a atualização é necessario somente as 6 primerias casas decimais. Clicando na locallização que o RN passou, voce sera redirecionado para o Google Maps, onde voce conseguirá pegar as coordenadas, como na imagem:



  2. Apos coletar as informações, deve ser carregada no arquivo Ajuste Coordenada - Semana X.xlsx.


    3. Voce devera adicionas os PDV e suas respectivas coordenadas no local indicado com o numero 1. Depois disso, será gerado, na parte indicada pelo numero 2, as coordenadas formatadas para importação.

  3. Em seguida, deverar copiar a coluna de exportação, para colocar no arquivo "GPS.CSV"

  4. Não é necessário abrir este arquivo no Excel. Basta clicar com o botão direito do mouse e selecionar "Editar". O Bloco de Notas será aberto, onde você poderá apagar o conteúdo antigo e colar as novas informações. Ao colar, você notará que automaticamente há uma "quebra de linha" para a próxima. Essa quebra deverá ser removida para evitar erros.

  5. Nos passos finais, copie o arquivo para a pasta "dvs". Dica: copiar o arquivo para a pasta é mais rápido do que tentar salvá-lo diretamente nela.

  6. Para finalizar, vá até a rotina 01.31.59 e insira o nome do arquivo seguido de ".CSV". Certifique-se de escrever corretamente, respeitando as letras maiúsculas. Basta colocar o nome do arquivo no local indicado, seguindo as instruções, e confirmar. Se tudo correr como esperado, uma mensagem de log confirmará a submissão.
