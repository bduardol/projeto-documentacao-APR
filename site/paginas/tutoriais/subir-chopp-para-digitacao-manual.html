Tutorial de digitação de chopp

Tutorial de como subir chopp.

Resumo

  1. O GV responsável deve enviar o preço e o PDV.
  2. Acesse a rotina (Promax) 01.25.08.03 para associar o PDV.
  3. Em seguida, acesse a rotina (Promax) 01.25.08.10 para verificar se há saldo disponível.
  4. O GV responsável deve inserir os dados necessários.
  5. Após concluir a inserção dos dados, o GV responsável deve informar que finalizou.
  6. Por fim, acesse a rotina 01.25.08.03 para remover o PDV, ou, alternativamente, faça o upload do template na rotina 01.25.08.09.

Detalhado

  1. O GV responsável deve enviar o preço e o PDV.

    2. O GV manda o valor do chopp. Com base no valor, basta ir na tabela de chopp, e ver qual codigo referente a marca e valor que ele mandou.

  2. Acesse a rotina (Promax) 01.25.08.03 para associar o PDV.

    3. No campo tabela escalonada, é o codigo referente da tabela, e cliente é o PDV que o GV mandou. Apos finalizar, basta salvar.

    É nesta mesma tela que é feito a retirada do PDV tambem. Basta segujir os mesmo passos, so que ao final, ao invez de salvar, deverá excluir

  3. Em seguida, acesse a rotina (Promax) 01.25.08.10 para verificar se há saldo disponível.
    4. Nem sempre tem necessidade de ir nesta rotina, mas é bom sepre conferir.
  4. Você deverar colocar a tabela, na qual op cliente esta cadastrado, e o cliente(PDV), tanto no final, quanto incial.
  5. Após consultar, voce poderar verificar o saldo do cliente para determinados chopps da tabela pesquisadas. Com isso, caso esteja zerado, poderar alterar o valor, para o GV responsavel poder digitar.

    6. Lembre-se: SEMPRE REMOVA A ASSOCIAÇÃO APOS O GV DIGITAR.
